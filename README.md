# NexusAI — Next-Gen AI Automation Platform

Hackathon submission for the Next-Gen AI Platform Speed Run.

## Features
- ✅ Feature 1: Matrix-Driven Pricing & Performance-Isolated Currency Switcher (INR/USD/EUR)
- ✅ Feature 2: Bento-to-Accordion with Context Lock on resize
- ✅ Zero external animation/UI libraries (WAAPI + native CSS)
- ✅ State-isolated currency/billing updates (DOM-only, no React re-renders)
- ✅ Full SEO + Open Graph metadata
- ✅ Responsive across all breakpoints
- ✅ All provided SVG assets integrated
- ✅ Syne + DM Sans fonts from the asset specification
